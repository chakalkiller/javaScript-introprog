Parcours : Javascript pour le web

Voici une série d'exercices orientés web. Le HTML et les CSS sont fournis. Vous devrez travailler dans le répertoire "src" où se trouvent les fichiers JS correspondant à chaque exercice.

Consignes :

  • Vous ne pouvez utiliser que var et let pour initialiser les variables.
  • Pas de onclick() dans le html mais des addEventListener dans le JS.
  • Pas de document.getElementBy... mais des querySelector.
  • Pas de Bootstrap ou autre librairie.
