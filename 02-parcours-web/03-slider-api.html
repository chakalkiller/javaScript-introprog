Javascript : Slider API

Exercice à développer dans src/03-slider-api.js

Sur base de l'exercice avec le slider, au lieu de récupérer les images en local, nous allons récupérer les images depuis une API (interface de programmation d'application).

Cette API fournit une image random lorsqu'on se connecte à Random Image API.

Il va falloir utiliser la méthode Fetch ainsi qu'une fonction asynchrone (via une promise) ainsi qu'apprendre à bien lire une documentation d'API.
