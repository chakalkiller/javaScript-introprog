Javascript : Méthode querySelector

Théorie

La méthode querySelector() renvoie le premier élément qui correspond à un sélecteur CSS.

Pour renvoyer toutes les correspondances (pas seulement la première), on utilise querySelectorAll().

querySelector() et querySelectorAll() renvoient une liste de noeuds.

On peut aussi utiliser Get Elements by ID, Tag, Name pour accéder au DOM, mais c'est moins précis que querySelector.
