Javascript : Variables

Théorie

Il y a plusieurs façons de déclarer des variables en JS.
  • var : var sert à définir une variable globale
  • let : let sert à définir une variable locale
  • const : const sert à déclarer une référence constante
  • rien

La portée (ou scope) d'une fonction est la seule à mettre toutes les variables sur un même pied d'égalité et déclarer une variable à l'intérieur d'une fonction n'aura logiquement pas d'incidence sur le reste des données du code.

Dans le JS moderne, on n'utilise plus que const et let!

Tout comme PHP, JS est un langage faiblement typé.
